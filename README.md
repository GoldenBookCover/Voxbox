# Voxbox

Voxbox 是一个基于 VoxCPM 模型构建的语音合成工具，提供简洁且功能强大的界面用于批量生成、参考音频克隆以及自动拼接音频内容。

## 特性

- **模型加载**：支持自定义模型路径及推理设备选择（CUDA, CPU, MPS 等）。
- **批量生成**：支持从文本框直接输入或通过 JSON 文件上传成批次处理，可避免逐条复制的操作。
- **参考克隆**：基于提供的 `reference_audio` 进行音色相似度对齐。
- **自动拼接**：可选开启音频缝合功能，在多个片段之间自动插入预设长度（如 0.5s）的静音。
- **定制配置**：支持调节 CFG Value、推理步数（Timesteps）、文本标准化等精细参数。
- **分享系统**：集成了内嵌 Web 服务，可在生成的页面中直接展示内容并播放音频（目前功能部分处于开发/预览阶段）。

## 快速开始

### 环境要求
- Python 3.10+
- CUDA 环境 (推荐用于高性能推理)
- 系统相关依赖库（如 ffmpeg）

### 安装步骤
1. 克隆仓库或[下载源码](https://github.com/GoldenBookCover/Voxbox/releases/latest)。
2. 自动下载集成环境：
```bash
.\run.bat
# 或在目录中双击执行
```

### 运行项目
在窗口中点击“启动主程序”，系统将自动在浏览器中打开对应的本地端口（默认 17865）。

## 参数说明
- **CFG Value**: 控制生成结果对提示词的跟随程度。
- **Inference Timesteps**: 采样步数，影响生成的细节质量与速度平衡。
- **Pause Duration**: 拼接多个段落时产生的静音间隔（单位：秒）。
- **Normalize**: 对输入文本自动处理符号和数字映射。

## 项目结构
- `src/main.py`: 主入口文件，包含 Gradio UI 和核心生成逻辑。
- `requirements.txt`: 项目所需依赖包列表。
- `reference_audio/`: 存放用于克隆的参考音频示例。
- `output/`: 生成文件的默认保存位置。

## 注释
本项目致力于简化 VoxCPM 系列模型的易用性，旨在为创作者提供高效、稳定的语音合成流程。
